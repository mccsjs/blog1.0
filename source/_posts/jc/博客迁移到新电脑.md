# 前言
## 因为怕自己忘了，写个教程记录一下

# 环境搭建
## node安装
官网下载最新版就行，目前最新的19版本没有不兼容问题[node官网链接](https://nodejs.org/en/download/)
可以更换淘宝源，速度更快
```SHELL
npm config set registry https://registry.npm.taobao.org
```

## 安装git
由于官网下载速度难顶，用淘宝源最新的[链接](https://registry.npmmirror.com/binary.html?path=git-for-windows/v2.36.1.windows.1/)
配置用户名和邮箱
```shell
git config --global user.name 用户名
git config --global user.email 邮箱
```
通过`git config -l`检查是否配置成功

## 安装HEXO
在Git BASH输入命令安装
```shell
npm install -g hexo-cli
```

# 连接Github
执行命令生成ssh公钥，连接Github
```SHELL
ssh-keygen -t rsa -C 邮箱
```
打开C盘下用户文件夹下的.ssh的文件夹，会看到 id_rsa.pub
用记事本打开id_rsa.pub，复制里面的内容，在github中配置ssh密钥。（设置-ssh and key）

通过以下命令确认是否连接成功
```shell
ssh -T git@github.com
```

出现连接到账户的信息就行

### 安装依赖
1.把我的源码解压下来后输入
```bash
npm i
```
进行初始化
2.安装 hexo-deployer-git
```shell
npm install hexo-deployer-git --save
```
3.现在已经完成了，hexo三连试试效果
```SHELL
hexo cl;hexo g;hexo d
```
